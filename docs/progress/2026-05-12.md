# Project Progress - 2026-05-12

## Accomplishments
1.  **Authentication Completion**
    *   Implemented JWT rotation with reuse detection (family-based revocation).
    *   Refresh tokens stored as SHA-256 hashes in DB.
    *   HttpOnly, Secure, SameSite=Strict cookies for refresh tokens.
2.  **Soft Delete Implementation**
    *   Added `deletedAt` to `User`, `Post`, and `Comment` models.
    *   Updated all service logic and middleware to filter for `deletedAt: null`.
3.  **Production-Ready Migrations**
    *   Transitioned from `db push` to `migrate dev`.
    *   Established clean migration baseline in `prisma/migrations`.
4.  **Caching Layer**
    *   Implemented `node-cache` utility.
    *   Added `cacheMiddleware` to automatically cache GET responses.
    *   Added cache invalidation logic to mutations (Create/Update/Delete).
5.  **Notifications & Alerts**
    *   Added `Notification` model for in-app user notifications.
    *   Created `AdminAlert` utility for real-time monitoring (Discord/Slack webhook ready).
    *   Integrated alerts for security events (token reuse) and registration events.
6.  **Post CRUD & Search**
    *   Implemented full Post service with slug auto-generation and read-time estimation.
    *   Added full-text keyword search across titles and content.
7.  **User Profiles & Comments**
    *   Implemented profile updates (bio, avatarUrl).
    *   Added nested commenting system with reply notifications.
8.  **Admin Insights**
    *   Created a stats dashboard for platform-wide metrics (users, posts, views).
9.  **API Documentation**
    *   Implemented Swagger/OpenAPI 3.0 documentation using `swagger-jsdoc`.
    *   Interactive UI available at `/api-docs`.
